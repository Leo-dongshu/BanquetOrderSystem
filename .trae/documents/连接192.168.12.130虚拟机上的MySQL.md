# 连接192.168.12.130虚拟机上的MySQL

## 现状分析
- 项目已经配置了连接192.168.12.130上的MySQL数据库
- 配置文件：`backend/src/config/database.ts`
- 数据库用户名：leo
- 数据库密码：leo123
- 数据库名称：banquet_order_system
- 项目已安装mysql2驱动（版本3.22.1）

## 连接步骤

### 1. 验证虚拟机MySQL服务状态
- 登录到192.168.12.130虚拟机
- 检查MySQL服务是否正在运行
- 确认3306端口是否开放

### 2. 验证数据库和用户权限
- 确保banquet_order_system数据库已创建
- 确保leo用户存在且有权限访问该数据库
- 验证用户密码是否正确

### 3. 测试连接
- 在本地运行后端服务
- 检查是否能够成功连接到MySQL数据库
- 查看服务启动日志，确认数据库连接状态

### 4. 故障排查
- 如果连接失败，检查网络连接是否正常
- 检查虚拟机防火墙设置
- 检查MySQL配置是否允许远程连接
- 验证用户名和密码是否正确

## 预期结果
- 后端服务能够成功连接到192.168.12.130上的MySQL数据库
- 数据库表结构能够自动创建
- 系统能够正常运行
